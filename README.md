# SIGA – Sistema Integrado de Gestão de Assinaturas

Projeto Integrador do curso de Engenharia de Software – UNISA (2026).

## Descrição
O SIGA é um protótipo acadêmico desenvolvido para simular o gerenciamento de clientes e assinaturas em um modelo de negócio por assinatura.

O sistema permite:
- Login de operador
- Listagem de clientes
- Visualização de detalhes
- Consulta de status de assinatura
- Logout

## Tecnologias Utilizadas
- Java
- Android Studio
- Arquitetura em Camadas
- Modelagem UML

## Objetivo Acadêmico
Aplicar conceitos de Engenharia de Software como:
- Levantamento de requisitos
- Modelagem UML
- Arquitetura de software
- Prototipação funcional
