# SIGA – Sistema Integrado de Gestão de Assinaturas

Projeto desenvolvido como parte do **Projeto Integrador** do curso de **Engenharia de Software – UNISA (2026)**.

---

## Sobre o Projeto

O SIGA é um protótipo funcional de um sistema mobile para gestão de assinaturas, permitindo:

- Login com autenticação simulada
- Listagem de clientes
- Visualização detalhada do cliente
- Exibição do status da assinatura
- Controle de sessão (login/logout)

A persistência de dados e autenticação foram simuladas por meio de repositórios fake, mantendo a arquitetura preparada para futura integração com **Firebase Auth** e **Cloud Firestore**.

---

## Tecnologias Utilizadas

- Java
- Android Studio
- RecyclerView
- Arquitetura em camadas
- Padrão Repository
- Simulação de Backend (Fake Repository)
